Sekkei's Website

Summer 2020
My friend and I co-founded a digital marketing agency during the summer of 2020 and since our work was almost entirely digital it was pretty much mandatory that we needed a website. I brought together 3 other friends who knew how to code and we worked together to bring it to life. I worked as the manager among my friends, tasking others with pages that needed to get done while I worked on the site structure, technology choices, and hosting. It was a great experience learning what it takes to keep a team productive through meetings and what goes into a modern website from start to finish.

A serverless website built with React and SCSS running on a netlify CMS and packaged with the gatsby compiler. Pretty FAST if u know what i'm talking about.
Try it: https://sekkeicreative.com

January 1, 2021
This project was a way for me to learn the full stack development experience, from server -> web -> domain registration -> vps -> deploy. It uses a next.js and typeorm front end with a node.js server using graphql to interface with a postgres database that is on a redis container. Following Ben Awad's tutorial (https://www.youtube.com/watch?v=I6ypD7qv3Z8) and building off of it. It's sort of a spiritual successor to Kylegram which was my clone of instagram but on a LAMP stack.
